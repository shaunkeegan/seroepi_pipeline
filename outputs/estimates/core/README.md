# outputs/estimates/core/

Summary posterior statistics for models fit to the full dataset (`core` mode). Generated by `02_model_process.R`. All files are excluded from version control.

## File naming convention

Files use opaque IDs: `YYYYMMDD-NNNNNN.csv` with a paired `YYYYMMDD-NNNNNN.json` sidecar.

Use `find_file_by_metadata()` in `R/utils.R` to locate files by antigen, subset, and count type.

## JSON sidecar metadata fields

| Field | Description |
|-------|-------------|
| `id` | Unique file ID (`YYYYMMDD-NNNNNN`) |
| `created` | ISO 8601 timestamp of file creation |
| `mode` | `"core"` |
| `antigen` | `"K"`, `"O"`, or `"KO"` |
| `subset` | `"Full"`, `"Carba"`, `"ESBL"`, or `"Fatal"` |
| `count_type` | `"adj"` or `"raw"` |
| `source_file` | Input site count filename |

## CSV columns

| Column | Description |
|--------|-------------|
| `locus` | Serotype locus identifier |
| `subgroup` | WHO region (Eastern Africa, Southern Africa, Southern Asia, Western Africa) |
| `mean` | Posterior mean prevalence (probability scale) |
| `median` | Posterior median prevalence |
| `lower` | Lower bound of 95% credible interval |
| `upper` | Upper bound of 95% credible interval |
