# data/

Input data for the serotype estimation pipeline. Files are excluded from version control (`.tsv`, `.csv`, `.rds`); this README ensures the directory structure is preserved in git.

## Subdirectories

| Directory | Contents |
|-----------|----------|
| `core/site_counts/` | Per-locus site count TSVs for the full dataset |
| `LOO/` | Per-locus site count CSVs with one study held out per file |

See the README in each subdirectory for file naming conventions and column descriptions.
